# 智能公元支持文档 (SmartPi Support Docs)

[![Docs Status](https://img.shields.io/badge/docs-maintained-brightgreen)](https://github.com/JX-t11/smartpi-support-docs)
[![License](https://img.shields.io/badge/license-Copyright-blue)](LICENSE)

本仓库托管了 [智能公元](https://smartpi.cn) 平台的官方技术文档、操作指南及开发资料。旨在帮助开发者快速上手智能公元模组、AI 智能体平台及相关开发工具。

## 📚 文档内容概览

文档体系涵盖以下核心板块：

### 1. 平台与工具
- **平台使用教程**: 固件配置全流程、参数详解、GPIO/串口控制及小程序操作指南。
- **智能体开发**: AI Agent 创建、知识库搭建、MCP 插件开发、API 服务发布及设备绑定。
- **开发指南**: 积木编程 (Blockly)、产品结构设计、开源仓库资源。

### 2. 模组资料详解
提供全系列模组的硬件规格、软件接口与开发资料：
- **离线语音模组**: SU 系列 (03T/10A/23T/63T等)、CI 系列、JX 系列。
- **AI 模组**: JX-A7T 等在线/离线 AI 芯片。
- **连接模组**: WiFi 图像模组 (XR-50A)、通用 WiFi 模组 (BL-62B, JX-12F)。

### 3. 技术支持与 FAQ
- **烧录与调试**: 各系列模组的烧录工具使用与故障排查。
- **语音调优**: 唤醒词灵敏度调整、识别效果优化指南。
- **硬件设计**: 原理图设计注意事项、PCB 布局建议。
- **应用案例**: 常见应用场景的实现方案与 Demo。
