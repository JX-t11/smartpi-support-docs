### 模组详细特性

#### CI-03T / CI-33T（离线语音模组）

- **脑神经网络处理器 BNPU V3**：支持 DNN/TDNN/RNN 等神经网络及卷积运算
- **音频接口**：双路 ADC + 单路 DAC，支持 8k~48kHz 采样率
- **外设接口**：3路 UART、1路 I2C、5路 PWM、9个 GPIO
- **电源管理**：内置 3 个 LDO，支持 5V 直接供电
- **版本区分**：

    - 普通双咪版本：不支持内部 AEC
    - 单咪 AEC 版本：支持内部 AEC 功能

#### SU-03T（离线语音模组）

- **处理器**：32位 RISC @240MHz，支持 DSP 指令集及 FPU 浮点运算
- **FFT 加速器**：最大支持 1024 点复数 FFT/IFFT
- **内置功放**：2.4W 单声道 AB 类功放
- **外设接口**：SPI Master/Slave、UART @3Mbps、I2C @400kHz、2路 PWM
- **ADC**：12-bit SAR-ADC @450kHz 采样率

#### JX-A7T（在线语音模组）

- **核心芯片**：XR872AT，ARM Cortex-M4F @384MHz
- **存储**：内置 4MB PSRAM
- **网络**：802.11b/g/n Wi-Fi
- **应用**：支持离线+在线混合语音识别，可接入智能体

#### BL-62B（WiFi 联网模组）

- **核心芯片**：BL602，32位 RISC CPU
- **网络**：2.4G Wi-Fi (802.11b/g/n) + BLE 5.0
- **外设接口**：UART、GPIO、ADC、DAC、PWM、I2C、SDIO、SPI、IR
- **电源管理**：支持多种低功耗模式
- **应用**：IoT 设备联网、智能家居控制

## 选型指南与建议
1. 按交互方式：离线（低延时）、在线（高自由度）、混合（兼顾）。
2. 按硬件资源：若主控资源有限可选择模块化方案，需深度定制时使用开放 SDK。
3. 按联网需求：需要视频选 XR-50A，只需语音控制可用 CI/SU + WiFi 组合。

## 应用场景推荐
- 家电：CI-03T + BL-62B。
- 智能音箱：JX-A7T + 定制功放。
- 工业告警：SU-23T + 小程序监控。
