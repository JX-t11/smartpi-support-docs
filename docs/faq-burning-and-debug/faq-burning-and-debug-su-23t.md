# SU-23T 烧录与调试 FAQ
 
### SU-23T烧录工具显示"等待设备上电"无法完成烧录怎么办？
 
**问题描述：**
 
使用US513U61 Production Tool烧录SU-23T模块时，工具界面显示"等待设备上电"，无法完成烧录。
 
**解决方案：**
 
    1. **硬件连接检查**：
 
        - 确认UART1的TX1/RX1引脚连接正确
        - 交叉连接：模块TX→转换器RX，模块RX→转换器TX
        - 确保接地连接良好
 
    2. **烧录工具配置**：
 
        - 下载模式选择：SIP-NOR-FLASH
        - 波特率：1000000（如失败可尝试115200）
        - 确保固件文件（.bin）完整且匹配
 
    3. **故障排查**：
 
        - 确认USB转串口驱动已正确安装
        - 检查COM端口是否被其他程序占用
        - 尝试重新上电模块
        - 更换USB端口测试
 
    **常见烧录方式参考**：
 
    - 脱机烧录器批量烧录
    - 串口烧录（需CH340转换芯片）
    - 开发板烧录（适合调试阶段）
 
    **注意事项：**
 
    - 确保引脚连接正确，避免混淆UART0和UART1
    - 烧录时单片机不要占用UART1
    - TX/RX需要交叉连接（模块TX → 烧录器RX；模块RX → 烧录器TX）
    - 确保GND共地、供电稳定，必要时缩短线材并避免接触不良
    - 固件路径尽量使用英文路径，避免中文/空格导致工具异常
    - 操作时注意防静电
    - 确保供电稳定，避免烧录中断
    - 使用与硬件版本匹配的固件

### SU-23T刚开机可以识别，过一会儿不能识别怎么办？

**问题描述：**

SU-23T模块在刚开机时语音识别功能正常，但工作一段时间后语音识别失效，无法响应唤醒词和命令词。

**可能原因与排查步骤：**

1. **深度睡眠设置问题**：

    - 在智能公元平台的「优化选项」中检查「深度睡眠」设置
    - 尝试关闭深度睡眠功能后重新烧录固件测试
    - 深度睡眠功能在某些批次可能存在兼容性问题，导致识别异常

2. **电源供电问题**：

    - 检查电源供电是否稳定，电压是否在规定范围（2.2-3.6V）内
    - 测试电源输出纹波和电流输出能力是否满足要求
    - 确认电源在长时间工作后是否出现压降或不稳定
    - 即使部分样品正常，也需要排查电源电路是否存在批次差异

3. **硬件一致性检查**：

    - 对比正常工作和异常的样品，检查硬件电路是否存在差异
    - 测量关键点的电压和波形，确认电源、时钟、复位等信号正常
    - 检查麦克风电路焊接和连接是否可靠

**注意事项：**

- 关闭深度睡眠后功耗会增加，需要权衡功耗和稳定性
- 如果问题仅出现在部分样品上，重点排查硬件批次差异
- 建议使用示波器监测电源稳定性，排查电源相关故障
