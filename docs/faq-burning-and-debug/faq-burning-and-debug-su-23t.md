# SU-23T 烧录与调试 FAQ
 
### SU-23T烧录工具显示"等待设备上电"无法完成烧录怎么办？
 
**问题描述：**
 
使用US513U61 Production Tool烧录SU-23T模块时，工具界面显示"等待设备上电"，无法完成烧录。
 
**解决方案：**
 
    1. **硬件连接检查**：
 
        - 确认UART1的TX1/RX1引脚连接正确
        - 交叉连接：模块TX→转换器RX，模块RX→转换器TX
        - 确保接地连接良好
 
    2. **烧录工具配置**：
 
        - 下载模式选择：SIP-NOR-FLASH
        - 波特率：1000000（如失败可尝试115200）
        - 确保固件文件（.bin）完整且匹配
 
    3. **故障排查**：
 
        - 确认USB转串口驱动已正确安装
        - 检查COM端口是否被其他程序占用
        - 尝试重新上电模块
        - 更换USB端口测试
 
    **常见烧录方式参考**：
 
    - 脱机烧录器批量烧录
    - 串口烧录（需CH340转换芯片）
    - 开发板烧录（适合调试阶段）
 
    **注意事项：**
 
    - 确保引脚连接正确，避免混淆UART0和UART1
    - 烧录时单片机不要占用UART1
    - TX/RX需要交叉连接（模块TX → 烧录器RX；模块RX → 烧录器TX）
    - 确保GND共地、供电稳定，必要时缩短线材并避免接触不良
    - 固件路径尽量使用英文路径，避免中文/空格导致工具异常
    - 操作时注意防静电
    - 确保供电稳定，避免烧录中断
    - 使用与硬件版本匹配的固件
