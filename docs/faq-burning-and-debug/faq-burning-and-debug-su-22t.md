# SU-22T 烧录与调试 FAQ

本页用于整理 SU-22T 相关的烧录与调试问题。

### 语音模块烧录后无法唤醒且无声音

**问题描述：**

模块可以正常烧录，但烧录后无法唤醒，喇叭也没有声音。

**解决方案：**

使用万用表按以下步骤排查模块是否损坏：

1. **短路检查**：万用表转到蜂鸣档，测量 VCC 和 GND 是否被击穿（短路）

2. **供电测试**：如果没有短路，给模块 5V 供电，用万用表测试：

    - 模块 3V3 引脚是否有 3.3V 电压输出
    - 麦克风正负电压是否在 1.6-1.7V 范围（CI-03T）或 2.4-2.5V 范围（SU-03T）
    - 喇叭正负对 GND 电压是否在 2.2-2.5V 范围内

**注意事项：**

- 如果某个测试电压不正常，可能是模块烧坏或异常
- SU-22T 模块 3V3 引脚正常输出应为 3.3V，如果测量到 4.6V 说明模块异常

### SU-22T和SU-21T的烧录稳定性对比如何？

**问题描述：**

SU-21T存在烧录不稳定的问题，询问SU-22T是否也会出现相同问题，以及两者的封装是否一致。

**解决方案：**

**型号对比：**

1. **封装差异**
    - SU-21T和SU-22T封装不同
    - 引脚定义可能有差异
    - 不能直接互换使用

2. **烧录稳定性**
    - SU-22T的烧录稳定性相对更好
    - 对连接质量要求相对宽松
    - 建议尝试SU-22T作为替代方案

**选择建议：**

- 如果项目对烧录稳定性要求高，可考虑使用SU-22T
- 更换型号前需确认功能需求匹配
- 两种型号的引脚定义需要重新核对

**注意事项：**

- 型号选择应基于功能需求，而非仅考虑烧录便利性
- 更换型号可能需要调整硬件设计
- 建议小批量测试后再批量采购
