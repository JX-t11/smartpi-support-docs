# SU-30T-31T 烧录与调试 FAQ

本页用于整理 SU-30T-31T 相关的烧录与调试问题。

### SU-30T1如何连接CH340进行烧录？

**问题描述：**

需要SU-30T1模块与CH340串口工具的连接图，以便正确进行固件烧录。

**解决方案：**

**1. 连接方法**

- **标准接线表**：

    - CH340的5V → SU-30T1的5V
    - CH340的RXD → SU-30T1的DB_TX（Pin_27）
    - CH340的TXD → SU-30T1的DB_RX（Pin_26）
    - CH340的GND → SU-30T1的GND
    - CH340的GND → SU-30T1的I04（Pin_24）

- **引脚说明**：

    - DB_RX/DB_TX为模块丝印标识
    - Pin_26/27是规格书上的引脚定义
    - I04（Pin_24）在烧录时需要连接到GND

**2. 操作步骤**

- **连接顺序**：

    1. 先断开所有电源
    2. 按照接线表连接各引脚
    3. 确认连接无误后给CH340上电
    4. 打开烧录工具选择固件开始烧录

- **烧录要点**：

    - 使用5V供电（非3.3V）
    - 确保所有地线连接良好
    - 烧录完成后断开I04与GND的连接

**注意事项：**

- 接线必须牢固，避免接触不良
- 不要将TX/RX线接到其他引脚
- 烧录时不要连接其他外设
- 如无法识别，检查COM端口驱动

### SU-30T烧录成功后命令词无响应怎么办？

**问题描述：**

SU-30T模块烧录成功后，能够被唤醒并应答，但说出命令词时没有反应，设备不执行相应动作。

**解决方案：**

**1. 操作流程确认**

- 必须先说出唤醒词将设备唤醒
- 唤醒成功后等待设备回应
- 清晰说出命令词

**2. 烧录软件确认**

- 使用云知声UniOne蜂鸟M烧录工具
- 确认烧录状态显示"下载完成"
- 烧录完成后重新上电测试

**3. 硬件检查**

- 确认使用的是SU-30T模块而非第三方产品
- 检查模块丝印标识是否正确
- 验证麦克风和喇叭连接

**注意事项：**

- 唤醒词和命令词是两步操作，不能连续说
- 烧录成功后需要等待设备完全启动
- 如使用第三方模块，请参考相应技术支持
- 确认命令词在固件配置中已正确设置
