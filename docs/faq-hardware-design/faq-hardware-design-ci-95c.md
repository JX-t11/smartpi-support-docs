# CI-95C 硬件设计 FAQ

本页用于整理 CI-95C 相关的硬件设计问题。

### CI-95C模块的供电电流为什么需要550mA？

**问题描述：**

CI-95C模块推荐供电电流为550mA，询问为何需要这么大的电流，以及是否可以降低要求。

**解决方案：**

550mA的供电电流要求是基于模块实际硬件需求确定的推荐值。

**电流需求分析：**

1. **音频功放需求**
    - 内置音频功放需要较大电流
    - 播放语音时瞬时电流较高
    - 保证音质和输出功率

2. **系统工作电流**
    - 芯片正常工作的基础电流
    - 语音识别处理的峰值电流
    - 各种外设的供电需求

3. **设计余量考虑**
    - 留有足够的工作余量
    - 确保在各种条件下稳定工作
    - 避免供电不足导致功能异常

**设计建议：**

1. **电源设计**
    - 按照550mA要求设计电源
    - 建议留有20%以上余量
    - 使用稳定的LDO或DCDC

2. **实际测试**
    - 正常工作电流约200-300mA
    - 峰值电流可能达到400mA以上
    - 需考虑最坏情况

3. **优化方案**
    - 如需降低电流，可考虑减少音量
    - 优化语音播报内容
    - 选择低功耗模式（如支持）

**注意事项：**

- 550mA是推荐值，不是恒定电流
- 实际电流根据使用情况变化
- 电源设计必须满足峰值需求
- 电流不足会导致重启或功能异常

### CI-95C2语音识别模组的备货和出货安排如何处理？

**问题描述：**

关于CI-95C2语音识别模组的备品情况，以及新一批2001块长虹模块的出货安排和合同签署状态。

**解决方案：**

**备品管理：**

1. **临时备品**：

    - 上次已额外寄送20个模块作为备品
    - 可临时应对生产需求
    - 建议合理规划备品使用

2. **批量生产安排**：

    - 新一批2001块长虹模块已安排生产
    - 预计10月25日出货
    - 需要提前确认合同签署

**出货流程：**

1. **合同管理**：

    - 销售合同已准备就绪
    - 需要双方签署后生效
    - 确保出货前完成所有手续

2. **生产计划**：

    - 与长虹协调生产排期
    - 确保按时交付
    - 预留充足的检验时间

**注意事项：**

- 备品数量有限，需合理使用
- 大批量采购建议提前规划生产周期
- 合同签署是出货的必要前提
