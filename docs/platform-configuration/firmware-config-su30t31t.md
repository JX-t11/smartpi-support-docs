# SU-30T-31T 固件配置

> 本文档专门针对 **SU-30T** 和 **SU-31T** 模组的固件配置，提供该模组特有的配置参数和注意事项。

## 模组信息

- **模组型号**：SU-30T / SU-31T
- **芯片型号**：US527U5
- **支持命令词数量**：150 条
- **麦克风配置**：双 MIC 离线语音模块
- **Flash 配置**：内置 8M 的 Flash、多引脚
- **特殊功能**：支持双 MIC 降噪和打断，支持高噪声环境
- **综合唤醒率**：98%

## 创建产品

在开始配置固件之前，需要先在智能公元平台创建对应的产品。以下是创建 SU-30T/SU-31T 产品的详细步骤：

### 1. 进入产品管理页面

1. 登录智能公元平台（[smartpi.cn](https://smartpi.cn)）
2. 在左侧导航菜单中，点击「产品管理」，展开子菜单后选择「所有产品」
3. 进入产品管理页面，可以看到产品列表和操作按钮

![产品管理页面](./img/product-manage-page-su30t31t.png)

### 2. 点击创建产品

在产品管理页面右上角，点击蓝色的「创建产品」按钮，进入产品创建向导。

### 3. 选择产品类别

产品创建向导分为三个步骤。第一步是选择产品类别：

1. 在「选择产品类别」步骤中，可以看到多个产品类别标签页（灯具、开关、插座等）
2. 对于 SU-30T/SU-31T 模组，通常选择「灯具」类别
3. 在灯具类别下，选择合适的子类型，例如：

    - **照明灯**：适用于普通照明场景
    - **RGB灯**：适用于彩色灯光控制
    - **台灯**：适用于桌面照明
    - 其他类型根据实际应用选择

![选择产品类别](./img/create-product-step1-su30t31t.png)

**示例**：选择「照明灯」类型，点击对应的卡片完成选择。

### 4. 选择场景和模组

第二步是选择应用场景和硬件模组：

#### 4.1 选择场景

在「选择场景/模组」步骤中，首先选择应用场景：

- **纯离线方案**：无需连网，离线也能操作（推荐用于 SU-30T/SU-31T）
- **Wi-Fi**：Wi-Fi产品，可远程控制产品
- **大模型**：离在线操作，可接入大模型智能体
- **在线**：随时随地可操作产品

![选择场景](./img/create-product-step2-scene-su30t31t.png)

**对于 SU-30T/SU-31T**：选择「纯离线方案」，因为 SU-30T/SU-31T 是离线语音识别模块。

#### 4.2 选择模组

选择场景后，页面会自动切换到「选择模组」标签页，显示可用的模组列表：

1. 在模组列表中查找并选择 **SU-30T (US527U5)** 或 **SU-31T (US527U5)**
2. 可以看到模组的详细信息：

    - 双MIC离线语音识别模块
    - 支持150条命令词
    - 内置8M的Flash、多引脚
    - 支持双MIC降噪和打断
    - 支持高噪声环境
    - 综合唤醒率98%

![选择模组](./img/create-product-step2-module-su30t31t.png)

3. 点击 SU-30T 或 SU-31T 模组卡片完成选择

### 5. 填写产品信息

第三步是填写产品的基本信息：

1. **产品名称**（必填）：

    - 输入一个有意义的产品名称，例如：「SU-30T智能照明灯」
    - 建议使用能清楚标识产品用途的名称

2. **选择语言**（必填）：

    - 从下拉菜单中选择语言，支持：中文、英文
    - SU-30T/SU-31T 支持中文和英文两种语言
    - 根据目标市场选择合适的语言

![填写产品信息](./img/create-product-step3-info-su30t31t.png)

3. 填写完成后，点击页面底部的「下一步」按钮

![完成产品信息填写](./img/create-product-step3-complete-su30t31t.png)

### 6. 产品创建完成

完成上述步骤后，系统会自动创建产品并跳转到「版本详情」页面，此时可以开始配置固件版本。

在版本详情页面，可以看到以下配置步骤：

- 前端信号处理
- Pin脚配置
- 唤醒词自定义
- 命令词自定义
- 发音人配置
- 其它配置
- 个性化音频
- 优化配置

## 版本配置详解

### 1. 前端信号处理

前端信号处理配置用于优化设备的语音识别效果，特别是针对双 MIC 配置的 SU-30T/SU-31T 模组。

![前端信号处理](./img/config-step1-frontend-signal-su30t31t.png)

**主要配置项：**

- **麦克风配置**：

    - **单MIC**：单麦克风配置
    - **双MIC**：双麦克风配置（SU-30T/SU-31T 推荐使用）
  
- **麦克风间距**：

    - 设置两个麦克风之间的距离（单位：mm）
    - 默认值：70mm
    - 根据实际硬件布局调整

- **距离识别**：

    - **近场0-1m**：适用于近距离语音识别
    - **远场1-5m**：适用于远距离语音识别（推荐）

- **AEC回声消除**：

    - 开启后可减少回声干扰
    - 根据实际应用场景选择是否开启

- **稳态降噪**：

    - 开启后可减少稳态噪声干扰
    - 根据实际应用场景选择是否开启

**注意事项：**

- SU-30T/SU-31T 支持双 MIC 配置，可提供更好的语音识别效果
- 双 MIC 配置需要正确设置麦克风间距
- 高噪声环境下建议开启 AEC 回声消除和稳态降噪

### 2. Pin脚配置

Pin脚配置用于定义 GPIO、PWM、UART 等端口的功能，实现语音指令控制外部设备。

![Pin脚配置](./img/config-step2-pin-config-su30t31t.png)

**主要配置项：**

- **Pin No.**：引脚编号
- **功能**：选择引脚功能（GPIO、PWM、UART、ADC 等）
- **参数1-4**：根据选择的功能配置相应参数
- **上电记忆**：是否保存上电时的状态
- **备注**：引脚功能说明

**注意事项：**

- SU-30T/SU-31T 模组提供多引脚支持，可根据实际硬件连接配置
- 注意引脚功能复用，避免冲突
- 参考 SU-30T/SU-31T 硬件资料中的引脚定义

### 3. 唤醒词自定义

唤醒词自定义用于设置设备的唤醒词和唤醒回复语。

![唤醒词自定义](./img/config-step3-wake-word-su30t31t.png)

**主要配置项：**

- **唤醒词**：

    - 输入自定义唤醒词（例如：「你好小美」）
    - 系统会自动生成对应的拼音
    - 可以添加多条唤醒词

- **唤醒灵敏度**：

    - **低**：唤醒灵敏度较低，误唤醒率低
    - **中**：平衡唤醒灵敏度和误唤醒率（推荐）
    - **高**：唤醒灵敏度较高，但误唤醒率可能增加

- **唤醒回复**：

    - 设置设备被唤醒后的回复语
    - 可以添加多条回复语，系统会随机选择
    - 例如：「我在」、「你说」、「有什么可以帮到你」

**注意事项：**

- 唤醒词建议使用 2-4 个汉字
- 避免使用容易误识别的词语
- 唤醒词和命令词不能重复

### 4. 命令词自定义

命令词自定义用于设置设备的语音控制命令和对应的回复语。

![命令词自定义（基础信息）](./img/config-step4-command-word-1-su30t31t.png)

**基础信息配置：**

- **序号**：命令词的序号
- **行为**：命令词对应的行为名称（例如：TurnOn、TurnOff）
- **触发方式**：选择「命令词」触发
- **命令词**：输入语音命令词，多条命令词用「|」分隔
- **回复语**：设备识别到命令词后的回复语

![命令词自定义（控制详情）](./img/config-step4-command-word-2-su30t31t.png)

**控制详情配置：**

- **控制类型**：选择控制类型（GPIO、PWM 等）
- **动作**：选择具体动作
- **动作参数**：配置动作参数

**其他配置：**

- **识别灵敏度**：

    - **低**：识别灵敏度较低，误识别率低
    - **中**：平衡识别灵敏度和误识别率（推荐）
    - **高**：识别灵敏度较高，但误识别率可能增加

- **免唤醒的命令词**：

    - 设置无需唤醒词即可直接识别的命令词
    - 唤醒词加上免唤醒的命令词，总数不能超过 10 条

- **防止误识别的命令词**：

    - 设置防止误识别的命令词
    - 不能和唤醒词、命令词重复
    - 多条词条之间用「|」隔开

**注意事项：**

- SU-30T/SU-31T 最大支持 150 条命令词
- 建议合理分配命令词，避免超出限制
- 命令词建议使用 2-6 个汉字
- 避免使用容易混淆的词语

### 5. 发音人配置

发音人配置用于设置设备的语音合成参数。

![发音人配置](./img/config-step5-speaker-su30t31t.png)

**主要配置项：**

- **合成音类型**：

    - **本地合成音**：使用本地语音合成（推荐）
    - **云端合成音**：使用云端语音合成

- **发音人**：选择语音合成发音人
    - 萱萱（甜美女生）
    - 玲玲（台湾女生）
    - 糖糖（活泼女生）
    - KiYo（可爱女生）
    - 小雯（女播音员）
    - 小峰（男播音员）
    - 天天（天真男生）

- **音量**：设置语音音量（0-100，默认 50）
- **语速**：设置语音语速（0-100，默认 50）
- **亮度**：设置语音亮度（0-100，默认 50）

**注意事项：**

- 可以根据应用场景选择合适的发音人
- 音量、语速、亮度参数可以根据实际需求调整
- 建议先试听效果再确定配置

### 6. 其它配置

其它配置用于设置设备的其他功能参数。

![其它配置](./img/config-step6-other-config-su30t31t.png)

**主要配置项：**

- **开机播报**：

    - **有开机播报**：设备开机时播放播报内容
    - **无开机播报**：设备开机时不播放播报
    - **播报内容**：设置开机播报的文字内容

- **超时退出**：

    - **超时时间**：设置超时退出时间（建议 5-60 秒，默认 10 秒）
    - **退出回复**：设置超时退出时的回复语

- **主动退出**：

    - **退出命令**：设置主动退出的命令词
    - **退出回复**：设置主动退出时的回复语

**注意事项：**

- 超时时间建议设置在 5-60 秒之间
- 退出命令和退出回复可以根据实际需求自定义
- 可以添加多条退出回复，系统会随机选择

### 7. 个性化音频

个性化音频用于上传自定义音频文件，替代平台自动合成的音频。

![个性化音频](./img/config-step7-personalized-audio-su30t31t.png)

**主要功能：**

- **上传音频文件**：为特定的词条上传自定义音频文件
- **音频管理**：管理已上传的音频文件
- **音频总大小**：显示已上传音频文件的总大小

**注意事项：**

- 音频文件格式需要符合平台要求
- 音频文件大小有限制，注意控制总大小
- 个性化音频可以提升用户体验

### 8. 优化配置

优化配置用于调整系统参数，优化系统性能。

![优化配置](./img/config-step8-optimization-su30t31t.png)

**主要配置项：**

- **优化选项**：选择优化选项（根据平台提供的选项选择）
- **语音指令总数**：显示当前配置的语音指令总数

**注意事项：**

- 阈值越大越敏感，但误识别率也会增高
- 未设置时自动取默认值
- 建议根据实际测试效果调整优化参数
