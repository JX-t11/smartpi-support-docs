# 在线语音模组 天猫精灵

## 定位与适用场景

- **定位**：接入天猫精灵生态的在线语音模组
- **适用场景**：

    - **智能家居**：语音控制灯光、家电、窗帘
    - **内容播报**：音乐、新闻、天气等内容服务
    - **在线问答**：AI 语音助手
    - **语音技能**：自定义技能扩展
- **云端依赖**：阿里云账号、天猫精灵控制台

## 模组概述

- 支持双/四麦阵列，远场唤醒
- 云端依赖：阿里云账号、天猫精灵控制台
- 网络要求：稳定的 2.4 GHz Wi-Fi，建议保证上行带宽与低延迟

### 支持模组

| 模组型号 | 说明 |
|---------|------|
| AM-01-S710 | 标准模组 Pro2 |
| AM-03-GSD630 | 高性能模组 |

---

## 硬件设计指南

1. **麦克风接口**：预留麦克风阵列接口，布线对称
2. **电源设计**：电源隔离语音前端与 Wi-Fi 射频，独立 LDO
3. **调试接口**：暴露 UART/JTAG 调试接口，便于烧录与日志采集
4. **SPI Flash**：预留足够的 Flash 存储空间

---

## 开发环境与工具

### 环境准备

1. 注册天猫精灵开放平台账号并创建产品
2. 下载配套 SDK、烧录工具、云端调试脚本
3. 准备微信/小程序账号用于扫码绑定体验

### 开发流程

1. 通过 USB/UART 烧录固件，确保供电稳定
2. 首次上电进入配网，手机小程序扫码绑定
3. OTA：在智能公元或天猫精灵后台上传新固件，配置灰度策略

---

## 外设开发与应用

### 语音技能配置

- **技能配置**：在控制台配置技能 → 下发至设备 → 通过语音触发
- **账号权限**：管理授权码、token 过期策略

### 典型开发流程

1. 上电初始化
2. 配网绑定
3. 技能配置
4. 测试语音命令
5. 云端反馈

### 网络与语音链路

| 参数 | 说明 |
|------|------|
| 音频编码 | 16 kHz，opus/pcm |
| 延时优化 | 本地唤醒词 + 云端意图分离 |
| 失败场景 | DNS、token 过期、云端超时 |
