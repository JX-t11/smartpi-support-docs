
# 在线语音模组 天猫精灵

## 文档定位与适用场景
- 适用于接入天猫精灵生态的语音模组或整机设备。
- 典型场景：智能家居控制、内容播报、在线问答、语音技能扩展。

## 模组概述与规格参数
- 支持双/四麦阵列，远场唤醒。
- 云端依赖：阿里云账号、天猫精灵控制台。
- 网络要求：稳定的 2.4 GHz Wi-Fi，建议保证上行带宽与低延迟。

## 硬件设计指南
1. 预留 SPI Flash、麦克风阵列、功放接口，布线对称。
2. 电源隔离语音前端与 Wi-Fi 射频，独立 LDO。
3. 暴露 UART/JTAG 调试接口，便于烧录与日志采集。

## 开发环境搭建与工具准备
- 注册天猫精灵开放平台账号并创建产品。
- 下载配套 SDK、烧录工具、云端调试脚本。
- 准备微信/小程序账号用于扫码绑定体验。

## 固件烧录与升级指南
1. 通过 USB/UART 烧录固件，确保供电稳定。
2. 首次上电进入配网，手机小程序扫码绑定。
3. OTA：在智能公元或天猫精灵后台上传新固件，配置灰度策略并监控日志。

## 外设开发与应用指南
- 语音意图：在控制台配置技能 → 下发至设备 → 通过语音触发。
- 账号与权限：管理授权码、token 过期策略，必要时批量刷新。
- 典型流程：上电 → 配网 → 技能配置 → 测试语音命令 → 云端反馈。

## 网络与语音链路
- 音频流：16 kHz，opus/pcm 编码。
- 延时优化：本地唤醒词 + 云端意图分离，配合 QoS。
- 失败场景：DNS、token 过期、云端超时，需在日志中打点。
