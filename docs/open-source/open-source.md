## 项目详情

### 01、模块转接板

- **适配模组**：SU-03T、SU-61/63T、CI-03T
- **功能**：模块烧录与测试的转接板 PCB
- **下载**：[模块转接板PCB优化 版本230220](https://help.aimachip.com/attach_files/kyck/455)

### 02、三极管开关电路模块

- **功能**：通用三极管开关电路模块
- **下载**：[三极管开关电路模块 版本221228](https://help.aimachip.com/attach_files/kyck/378)

### 03、智能语音茶吧机

- **适配模组**：SU-61T
- **功能**：智能语音控制茶吧机示例
- **下载**：[固件包](https://help.aimachip.com/attach_files/kyck/489)

### 04、带打断功能蓝牙音箱

- **适配模组**：CI-03T
- **功能**：支持语音打断的蓝牙音箱方案
- **下载**：[固件 230213](https://help.aimachip.com/attach_files/kyck/490)

### 05、语音控制抖音

- **适配模组**：CI-03T
- **功能**：语音控制抖音上下滑和视频双击点赞
- **下载**：[固件包](https://help.aimachip.com/attach_files/kyck/497)

### 06、卡针转接板

- **功能**：适用于小批量模块烧录和测试
- **下载**：[模块转接板 版本230404](https://help.aimachip.com/attach_files/kyck/513)

## 项目获取与编译

1. 下载对应项目资源包
2. 根据 README 配置工具链（GCC/Keil/Arduino IDE）
3. 使用官方烧录工具完成固件下载

---

## 贡献代码指南

- 提交前运行格式化、lint、单元测试
- PR 中说明适配模组、测试环境、影响范围
- 遵循仓库分支策略与 issue 模板
