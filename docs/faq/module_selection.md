# 模块选型与功能能力 FAQ

### 新生成的固件能下载到老的产品上吗？

可以的。固件继承重新发布不会产生影响，模块的引脚功能也没有变动。

---

### 需要获取CI-03T、CI-33T以及蓝牙模块的资料，包括文字版的操作流程和步骤说明。

CI-03T、CI-33T和SU-63T的资料链接可在官网获取，资料内包含原理图、模块和芯片技术手册、接线、烧录软件、烧录资料以及开发包。目前蓝牙产品主推SU-63T，新款CI-86Z的米思齐（Mixly）支持暂未上线。
