# SU-13T 应用场景与项目 FAQ

本页用于整理 SU-13T 相关的应用场景与项目问题。

### SU-13T是否支持4G远程和摄像头功能？

**问题描述：**

需要在SU-13T设备上实现4G远程通信、摄像头监控、温度显示和参数输入等功能。

**解决方案：**

**功能支持情况**

SU-13T模块目前**不支持**以下功能：

- **4G远程通信**：无内置4G模组接口
- **摄像头连接**：无视频处理能力
- **网络视频传输**：不支持视频流处理
- **远程监控界面**：无法实现摄像头画面传输

**技术限制原因**

- SU-13T是纯离线语音识别模块
- 处理器资源主要用于语音处理
- 无足够的算力处理视频数据
- 缺少网络通信硬件接口

**替代方案建议**

如需要远程监控功能，可考虑：

1. **多模块方案**：

    - SU-13T负责语音控制
    - 增加独立的4G模块和摄像头模块
    - 通过MCU协调各模块工作

2. **选择其他型号**：

    - JX-12F：支持WiFi和蓝牙
    - SU-63T：支持无线连接
    - 但仍不支持视频处理功能

3. **外接方案**：

    - 使用带WiFi功能的摄像头
    - 通过网络模块传输数据
    - SU-13T仅作为语音控制端

**注意事项：**

- SU-13T设计初衷是离线语音识别
- 视频处理需要专门的硬件支持
- 如需完整IoT方案，建议选择更强大的平台
