## 学习资源

### 有CI系列芯片的硬件小项目开发课程吗？

**问题描述：**

寻找针对CI系列芯片的硬件小项目开发课程或学习资源。

**解决方案：**

**1. 官方学习资源**

- **智能公元文档中心**：

    - 提供所有模块/芯片的完整资料
    - 包含图文指导的开发教程
    - 开发包内有详细的芯片原理图
    - 模块烧录操作视频

- **B站官方频道**：

    - 搜索UP主"机芯智能"
    - 包含指导视频、案例和教学
    - 从入门到进阶的完整教程

- **智能公元平台"发现"栏目**：

    - 平台入门视频教程
    - 模块烧录操作演示
    - 实际应用案例分析

**2. 学习路径建议**

- **基础入门**：熟悉平台操作和模块选型
- **实践操作**：跟随视频完成第一个项目
- **进阶学习**：参考开发包中的案例
- **项目实战**：结合实际需求开发应用

**注意事项：**

- 所有官方教程都是免费的
- 建议从简单功能开始逐步学习
- 遇到问题可查看FAQ或联系技术支持
