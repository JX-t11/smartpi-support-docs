# SU-30T-31T 模块选型 FAQ

本页用于整理 SU-30T-31T 相关的模块选型问题。

### SU-31T停产后的替代方案

**问题描述：**

SU-31T已停产，需要了解替代型号以及如何将现有SU-03T平台的语音指令快速迁移到新平台。

**解决方案：**

**替代型号选择**

1. **SU-32T**：

    - SU-31T的正式替代型号
    - 双麦克风配置，提供更好的降噪和唤醒性能
    - 识别率更高，适合嘈杂环境
    - 支持更多的GPIO和功能接口

2. **双麦克风替代方案**：

    - CI-03T4：双麦版本，但效果不如SU-32T
    - 如追求最佳唤醒效果，推荐选择SU-32T

**平台迁移方法**

1. **命令词复制**：

    - 使用命令词编辑器复制原有命令词
    - 可批量导出命令词列表
    - 在新平台中批量导入

2. **重新配置建议**：

    - 平台不支持直接导入配置文件
    - 需要在新平台中重新创建产品
    - 逐项重新配置功能逻辑

3. **唤醒性能提升**：

    - SU-32T的双麦克风设计相比单麦方案有显著提升
    - 理论上可以实现更灵敏的唤醒效果
    - 需要根据具体环境调试优化

**注意事项：**

- SU-31T和SU-32T引脚兼容，硬件设计无需修改
- 平台配置需要重新开始，无法直接迁移
- 建议在切换前记录完整的配置信息
- 保留原有命令词文本便于快速重建
