# 模块选型 FAQ

### 产品系列对比

**问题描述：**

需要了解各产品系列的特点和差异，以便选择合适的模块。

**各产品系列特点：**

**SU系列** - 离线语音基础款：

- 爆款产品，性价比高
- 低功耗设计，适合电池供电设备
- 多引脚资源，便于功能扩展
- 支持红外遥控功能
- 支持蓝牙音乐传输（部分型号）
- 支持小程序控制

**CI系列** - 离线语音增强款：

- 支持自学习功能，可训练自定义命令词
- 支持语音打断（Barge-in）功能
- 支持声纹识别，可区分不同用户
- 内置降噪算法，提升识别准确率
- 支持TTS文本转语音功能
- 支持红外遥控功能
- 支持小语种定制

**JX系列** - 智能网络款：

- 支持WiFi联网功能
- 支持微信小程序控制
- 集成AI大模型能力
- 支持4G网络通信（部分型号）
- 适合需要联网的智能应用场景

**选型建议：**

1. **基础离线语音控制** → 选择SU系列
2. **需要高级语音功能** → 选择CI系列
3. **需要联网/AI功能** → 选择JX系列

详细技术文档可参考：https://help.aimachip.com/docs/jx12f

### 离线语音控制与AI在线对话的区别

**问题描述：**

需要了解离线语音控制和AI在线对话两种方案的差异，以便根据产品需求选择合适的方案。

**方案对比：**

| 特性 | 离线语音控制 | AI在线对话 |
|------|-------------|-----------|
| **工作方式** | 必须预先设置好指令才会触发 | 可以进行AI对话调用大模型 |
| **网络依赖** | 无需联网，本地处理 | 需要联网连接云端服务器 |
| **响应速度** | 毫秒级响应，延迟极低 | 取决于网络状态，通常数百毫秒 |
| **识别能力** | 只能识别预设的固定指令 | 支持自然语言理解，灵活性高 |
| **指令数量** | 受模块内存限制（50-500条不等） | 理论上无限制，取决于大模型 |
| **隐私安全** | 数据本地处理，隐私性好 | 数据上传云端，需注意隐私 |
| **功耗** | 低功耗，适合电池供电 | 联网功能增加功耗 |
| **成本** | 模块成本较低，无服务器费用 | 需要支付服务器/API调用费用 |

**适用场景：**

**离线语音控制适合：**
- 智能家居设备（灯具、开关、窗帘等）
- 需要快速响应的语音控制
- 无网络环境或网络不稳定的应用
- 对隐私保护要求较高的场景
- 电池供电的低功耗设备

**AI在线对话适合：**
- 智能音箱、语音助手
- 需要自然语言交互的产品
- 需要获取实时信息（天气、新闻等）
- 需要复杂对话逻辑的应用
- AI玩具、教育机器人等

**离在线结合方案：**
- JX-A7T等模块支持离在线双模
- 本地快速响应常用指令，联网后调用AI大模型
- 兼顾响应速度和智能体验

**选型建议：**

1. **简单控制需求** → 纯离线方案（CI/SU系列）
2. **智能对话需求** → 纯在线方案或离在线结合（JX系列）
3. **兼顾响应和智能** → JX-A7T离在线双模方案

### CI13241(1MB)与CI13242(2MB)如何选择？

**问题描述：**

在噪声环境下使用语音识别时，CI13241(1MB Flash)的识别率不理想，需要了解是否需要更换为CI13242(2MB Flash)。

**解决方案：**

**核心差异：Flash容量与抗噪能力**

| 参数 | CI13241 (1MB) | CI13242 (2MB) | 选型建议 |
|------|--------------|--------------|----------|
| Flash容量 | 1MB | 2MB | 根据固件需求选择 |
| 抗噪能力 | 较弱 | 较强 | 噪声环境选2MB |
| 支持的模型 | 基础模型 | 高噪声模型（油烟机等） | 特定场景需2MB |
| 语音指令条数 | 100条 | 100条 | 相同 |

**选型建议：**

1. **安静/中噪声环境**：CI13241(1MB)即可满足需求
2. **高噪声环境**（如油烟机、工厂设备）：优先选择CI13242(2MB)
3. **需要专用高噪声模型**：必须使用2MB版本，1MB版本无法容纳此类模型

**技术说明：**

- 1MB Flash的上限限制了高噪声算法模型的存储
- 2MB Flash可以支持更大、更复杂的降噪模型
- 对于电机、风扇等噪声源明显的应用场景，2MB版本抗噪效果更佳

**注意事项：**

- 如果在噪声环境下识别率不理想，单纯调整阈值效果有限，建议更换2MB版本
- 更换芯片后需重新生成并烧录对应的固件
- 成本差异较小，建议在噪声不确定的情况下预留2MB方案

    - 训练周期和费用根据项目需求确定
- 实际效果需要现场测试验证

### CI-33T1与SU-23T如何进行选型对比？

**问题描述：**

在项目选型时需要在CI-33T1和SU-23T两个模块之间做出选择，需要了解两者的主要区别和适用场景。

**解决方案：**

**核心差异对比：**

| 特性 | CI-33T1 | SU-23T | 选型建议 |
|------|---------|--------|----------|
| **芯片平台** | CI1303 | CI1303 | 底层芯片相同 |
| **功耗特性** | 标准功耗（待机约60mA） | 超低功耗（待机约1mA） | 电池供电选SU-23T |
| **Flash容量** | 4M | 2M | 需要大容量选CI-33T1 |
| **语音指令条** | 500条 | 150条 | 复杂控制选CI-33T1 |
| **供电电压** | 3.6-5.5V | 2.2-3.6V | 宽电压选CI-33T1 |
| **内置功放** | 支持（最大2.4W） | 内置0.62W功放 | 大功率输出选CI-33T1 |

**主要技术特点：**

**CI-33T1优势：**

- 支持更多语音指令（500条 vs 150条）
- Flash容量更大（4M vs 2M）
- 支持更宽电压范围
- 支持更复杂的算法功能
- 适合功能复杂的交互应用

**SU-23T优势：**

- 极低功耗设计，待机电流约1mA
- 支持2.2-3.6V宽电压范围
- 适合电池供电设备
- 可支持两节干电池供电
- 深度休眠模式下约800μA

**典型应用场景：**

**选择CI-33T1的场景：**

- 市电供电的智能家电
- 需要复杂语音交互的产品
- 需要超过150条命令词的应用
- 需要大Flash存储语音文件
- 中等噪声环境使用

**选择SU-23T的场景：**

- 电池供电的便携设备
- 对功耗有极端要求的产品
- 可穿戴设备
- 单火开关等低功耗应用
- 简单语音控制场景

**成本考量：**

- CI-33T1：功能更全面，单价相对较高
- SU-23T：低功耗优化，性价比高
- 具体价格需联系销售获取最新报价

**注意事项：**

- SU-23T严禁5V供电，只能使用3.3V
- 两者引脚可能不兼容，设计时需注意
- 低功耗版本在功能上相对简化
- 建议根据实际功耗需求进行选型

### 系统只支持数字麦克风时如何选型？

**问题描述：**

主控平台的ADC芯片变更，系统只支持数字麦克风（Digital Mic/DMIC）输入，需要选择兼容的语音模块。

**解决方案：**

**数字麦克风支持情况：**

目前大部分语音模块使用模拟麦克风接口，支持数字麦克风的型号有限：

| 型号 | 数字麦支持 | 接口类型 | 备注 |
|------|-----------|---------|------|
| CI-73T | 支持 | 数字麦/SDIO/PCM | 支持数字麦克风输入 |
| CI-03T | 部分 | I2S/PDM | 支持PDM接口数字麦，但需确认具体型号 |
| JX-A7T | 支持 | I2S | 支持I2S接口连接麦克风阵列 |

**选型建议：**

**1. 首选方案：CI-73T**

- 支持数字麦克风接口
- 可直接连接PDM/I2S数字麦克风
- 无需额外的模数转换电路

**2. 备选方案：外接ADC芯片**

- 在模拟麦克风模块和数字麦之间添加ADC转换芯片
- 将模拟信号转换为数字信号后输入
- 增加硬件成本和复杂度

**3. 更换主控平台**

- 如对数字麦无强制要求，可考虑支持模拟麦克风的主控
- 大多数语音模块支持模拟麦克风，选型范围更广

**注意事项：**

- 数字麦克风方案成本通常高于模拟麦克风
- 数字麦需要确认具体的通信协议（I2S、PDM等）
- 建议申请样品进行实际测试验证
- 特殊需求可咨询技术支持确认兼容性

### 有没有比 SU-32T 灵敏度/识别率更好的模块？

**问题描述：**

在使用 SU-32T 模块后，希望了解是否有灵敏度或识别率更高的模块可供选择。

**解决方案：**

**SU-32T 是目前离线语音模块中识别率最高的型号。**

| 模块型号 | 综合识别率 | 适用环境 | 备注 |
|---------|-----------|---------|------|
| SU-32T | **98%** | 嘈杂高噪声环境 | 最高识别率，双麦降噪 |
| CI-03T系列 | 95% | 中噪声环境 | 功能丰富，支持AEC |
| SU-21T/22T/23T | 93% | 便携/穿戴设备 | 低功耗优化 |
| SU-03T | 90% | 中噪声环境 | 经典款，性价比高 |

**核心结论：**

- **SU-32T 已是当前离线语音模块的最高识别率规格（98%）**
- 专为高噪声环境优化，采用双麦克风阵列设计
- 如需进一步提升识别效果，可考虑以下方向：

**优化建议：**

1. **硬件优化**
   - 调整麦克风安装位置，避免噪声源直吹
   - 使用防风罩或防水咪头减少环境干扰
   - 确保麦克风供电稳定，避免电源噪声

2. **固件调优**
   - 在智能公元平台调整唤醒阈值
   - 优化词条发音，避免相似词条
   - 使用双麦降噪算法（SU-32T已支持）

3. **环境改善**
   - 减少背景噪声源
   - 避免强风或强气流环境
   - 增加模块与用户的距离范围

4. **在线语音方案**
   - 如需求远超离线识别能力，可考虑 JX-A7T 等离在线结合方案
   - 在线语音可接入AI大模型，识别能力更强

**注意事项：**

- SU-32T 需 4.5-5.5V 供电，设计时注意电源匹配
- 双麦版本需要正确的麦克风布局才能发挥最佳效果
- 如识别率仍不理想，建议申请样品测试或联系技术支持进行场景评估
