# JX-B5C 模块选型 FAQ

本页用于整理 JX-B5C 相关的模块选型问题。


### JX-B5C2模块IO引脚不足时如何选择支持物理按键的替代模块？

**问题描述：**

使用JX-B5C2模块开发语音控制蓝牙音响时，发现IO引脚不足，无法连接物理按钮，需要寻找其他支持物理按键功能的模块。

**解决方案：**

JX-B5C2模块的IO引脚资源非常有限：

- **可用引脚**：仅TX和RX两个通信引脚
- **其他引脚**：VCC、GND、喇叭输出和咪头输入
- **GPIO数量**：无额外的通用IO口

**推荐替代方案：SU-63T**

当需要连接物理按键时，推荐使用SU-63T模块：

- **丰富的IO资源**：提供多个GPIO引脚，支持连接物理按键
- **语音控制功能**：完整的语音识别和控制能力
- **蓝牙音乐功能**：支持蓝牙播放音乐，可语音控制切换歌曲
- **开发资源齐全**：提供完整的开发包和技术文档

**选型对比：**

| 特性 | JX-B5C2 | SU-63T |
|------|---------|---------|
| IO引脚 | 仅TX/RX | 多个GPIO |
| 物理按键 | 不支持 | 支持 |
| 语音控制 | 支持 | 支持 |
| 蓝牙音乐 | 支持 | 支持 |
| 开发资源 | 基础 | 完整开发包 |

**注意事项：**

- JX-B5C2主要用于简单的语音控制场景，不适合需要外接多个设备的应用
- SU-63T虽然IO资源更丰富，但两模块的硬件和软件不兼容，不能直接替换
- 选型时需综合考虑项目需求，如果需要物理按键控制，应优先选择SU-63T
