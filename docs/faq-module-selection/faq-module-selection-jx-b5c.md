# JX-B5C 模块选型 FAQ

本页用于整理 JX-B5C 相关的模块选型问题。


### JX-B5C2模块IO引脚不足时如何选择支持物理按键的替代模块？

**问题描述：**

使用JX-B5C2模块开发语音控制蓝牙音响时，发现IO引脚不足，无法连接物理按钮，需要寻找其他支持物理按键功能的模块。

**解决方案：**

JX-B5C2模块的IO引脚资源非常有限：

- **可用引脚**：仅TX和RX两个通信引脚
- **其他引脚**：VCC、GND、喇叭输出和咪头输入
- **GPIO数量**：无额外的通用IO口

**推荐替代方案：SU-63T**

当需要连接物理按键时，推荐使用SU-63T模块：

- **丰富的IO资源**：提供多个GPIO引脚，支持连接物理按键
- **语音控制功能**：完整的语音识别和控制能力
- **蓝牙音乐功能**：支持蓝牙播放音乐，可语音控制切换歌曲
- **开发资源齐全**：提供完整的开发包和技术文档

**选型对比：**

| 特性 | JX-B5C2 | SU-63T |
|------|---------|---------|
| IO引脚 | 仅TX/RX | 多个GPIO |
| 物理按键 | 不支持 | 支持 |
| 语音控制 | 支持 | 支持 |
| 蓝牙音乐 | 支持 | 支持 |
| 开发资源 | 基础 | 完整开发包 |

**注意事项：**

- JX-B5C2主要用于简单的语音控制场景，不适合需要外接多个设备的应用
- SU-63T虽然IO资源更丰富，但两模块的硬件和软件不兼容，不能直接替换
- 选型时需综合考虑项目需求，如果需要物理按键控制，应优先选择SU-63T

### JX-B5C2模块是否支持红外功能？

**问题描述：**

需要确认JX-B5C2蓝牙+语音识别模块是否支持红外收发功能，用于控制空调等红外设备。

**解答：**

**JX-B5C2模块不支持红外功能。**

**功能说明：**

JX-B5C2是蓝牙+语音识别二合一模块，主要功能包括：

- **语音识别**：离线语音唤醒和命令识别
- **蓝牙音频**：蓝牙音乐播放和手机通话
- **音频接口**：麦克风输入和喇叭输出
- **串口通信**：TX/RX串口控制接口

**不包含的功能：**

- **红外发射**：无红外发射功能
- **红外接收**：无红外接收功能
- **GPIO控制**：无可用通用IO口

**如需红外功能的替代方案：**

| 功能需求 | 推荐模块 |
|---------|---------|
| 语音+红外控制 | SU-03T、CI-03T |
| 语音+红外+蓝牙 | CI-95C、CI-96Z |
| 语音+物理按键+红外 | SU-63T |

**注意事项：**

- JX-B5C2硬件设计未集成红外收发电路
- 平台软件未提供红外功能配置选项
- 如项目需要红外控制功能，建议在选型阶段选择支持红外的模块
